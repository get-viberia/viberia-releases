# Viberia

> Think SimCity, but for AI teams.

Plan, staff, and scale your AI city. Dev, Design, Ops — all on one map.
Local-first. BYO keys and subscriptions.

## System Requirements

- macOS 13 (Ventura) or later
- Apple Silicon (M1+) or Intel

## Installation

1. Download the `.dmg` file from the [latest release](../../releases/latest)
2. Open the `.dmg` and drag Viberia to your Applications folder
3. Launch Viberia from Applications
